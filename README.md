## Run Locally

Clone the project

```bash
  git clone https://github.com/amr8644/Hostel-Management-System
```

```bash
  cd Hostel-Management-System

```

For C

```bash
  gcc main.c linkedlist.c
```

For Python

```bash
  py main.py
```
